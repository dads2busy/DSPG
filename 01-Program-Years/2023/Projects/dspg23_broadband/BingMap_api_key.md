# Steps to get an educational Bing Map API key

Education API key allows for free access of 50,000 transactions per 24 hours, where as the basic API key allows for 125,000 annually.

## 1. Create Bing Map account
Use this link to create your Bing Map account: https://www.bingmapsportal.com/.

Click "sign in" -> Login with Microsoft Account (associated with @virginia.edu) -> enter your information and create Bing Map account.

## 2. Create a Basic API key
Once logged, clicked on "My Account" menu -> click on "My Keys", then follow "Click here to create a new key."

Enter the application name (any non-empty string should be fine), and make sure "key type" is Basic and "Application type" is Dev/Test, then click on "Create".

## 3. Upgrade Basic key to Education.
Email to _mpnet@microsoft.com_ with your API key, the account ID (can be found under "My account" -> "Account details"), and a brief description of the project. Make sure to address which one of the following categories your project fall in:

- (i) publicly available without restriction (for example, login or password must not be required);
- (ii) available only to current students of your educational organization via your private network in order to provide education-related services; or
- (iii) available internally for free instructional use or available internally for non-commercial research use.

The complete requirement to be qualified as educational can be found here https://www.microsoft.com/en-us/maps/product#section-pst-limited-license under Section 3(b).

## 4. Wait for Microsoft's reply
Once you receive their reply email, your api key should have been upgraded. For your information, their response time is

|------------------------------|------------------------|-------------------------|
|     Pacific Standard Time    |     Before 12:00pm     |     Same Day            |
|     Greenwich Meantime       |     Before 8:00pm      |     Same Day            |
|     Pacific Standard Time    |     After 12:00pm      |     Next Day            |
|     Greenwich Meantime       |     After 8:00pm       |     Next Day            |
