Brief Overall Description of the Dataset:

Planet Labs provides low-cost imaging satellites at 3-5m resolution, known as Doves, and were first launched in 2013 . aim is to launch an “estimated magic number of 150 satellites” -- the largest armada in orbit -- to create an Earth observation satellite constellation with open data access. Satellite imagery from the Dove satellites is delivered via API and web tools. Before being viewed, each image is processed through an automated data pipeline. API users can then integrate Planet Labs imagery data into the their products, workflows, and analyses.

Link:  https://www.planet.com/
