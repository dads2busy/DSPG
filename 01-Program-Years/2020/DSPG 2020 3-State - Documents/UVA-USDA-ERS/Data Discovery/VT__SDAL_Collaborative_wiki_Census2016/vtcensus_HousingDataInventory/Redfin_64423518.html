VT Census Case Studies : Redfin

Created and last modified by molfe on 02 Jan 2016

Brief Overall Description of the Dataset:

Redfin is a residential real estate company that provides an online real estate database and brokerage services. Included in the database are all “MLS listings unless the seller requests that his listing not be published to the Internet. If a site is not run by a broker, it displays a partial set of listings, obtained through side-deals with individual brokers.” Also included are foreclosure and for-sale-by-owner. The data shown includes MLS data (i.e. square footage and number of homes) alongside of historical data (past selling and prices).

Link: https://www.redfin.com/
