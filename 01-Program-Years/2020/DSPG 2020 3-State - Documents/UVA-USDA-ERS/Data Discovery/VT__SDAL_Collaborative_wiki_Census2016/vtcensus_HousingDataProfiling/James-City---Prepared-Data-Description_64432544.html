VT Census Case Studies : James City - Prepared Data Description

Created by gkorkmaz, last modified by molfe on 31 Dec 2015

This page provides basic descriptions of the key variables used in this study after they were prepared.

Steps followed:

  • STEP 1 - Take unique rows
  • STEP 2 - Remove rows for which Field No. 93 == "DR: Duplicate record"
  • STEP 3 - Choose residential units, Standardized Land Use Code (V73) == "1001"  "1008"  "1110" "8001"
           - 1001 ---> 502 (V72) - SINGLE FAMILY - SUBURBAN
           - 1008 ---> 105, 106 (V72) - AGRICULTURAL 100+ ACRES WITH RESIDENCE, AGRICULTURAL 20-99 ACRES WITH RESIDENCE
           - 1110 ---> 503 (V72) - MULTI-FAMILY
           - 8001 ---> 502, 503 (V72) - SINGLE FAMILY - SUBURBAN & MULTI-FAMILY
  • STEP 4 - Remove parcels with assessed improvement value == 0
  • STEP 5 - Remove parcels with assessed land value == 0
