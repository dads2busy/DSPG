VT Census Case Studies : Arlington - Prepared Data Description

Created by gkorkmaz, last modified by molfe on 31 Dec 2015

This page provides basic descriptions of the key variables used in this study after they were prepared.

Steps followed:

1. Recode Land Use codes (V72) and select the Residential Land Use Codes: "Condo" "Single Family - Detached" "Single Family - Attached" "Multifamily" "Unknown Affordable Dwelling Unit" "MixedUsed"

2. Remove the parcels with zero Assessed Improvement Value (V38)

3. Remove the parcels with zero Assessed Land Value (V37)

4. Select the parcels with Assessed Land Value (V37) greater than 15,000.
