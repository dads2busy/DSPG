Home Appreciation

Data Set: Tract

Variable: APP_1Q

Description: Home appreciation last quarter

Rent 

Data Set: Tract

Variable: RENT_STCLS

Description: Class within State for Rent (10=highest)

Quiet Score

Data Set: Tract

Variable: QUIET_SCORE

Description: Quiet Score (100=best match)

Walk Score

Data Set: Tract

Variable: WLK_SCORE

Description: Quiet Score Walkability Score (100=best match)

 

Crime Index (compared to US)

Data Set: Tract

Variable: CRIME_INDEX_US

Description: Total Crime Index relative to Nation (0-5,000, 100 is average)

Crime Index (compared to state)

Data Set: Tract

Variable: CRIME_INDEX_STATE

Description: Total Crime Index relative to State (0-5,000, 100 is average)

Crime Index (compared to county)

Data Set: Tract

Variable: CRIME_INDEX_COU

Description: Total Crime Index relative to County (0-5,000, 100 is average)

Violent Crime (count)

Data Set: Tract

Variable: COUNT_VIOLENT

Description: Number of Violent Crimes

Violent Crime (per 1,000 population)

Data Set: Tract

Variable: PT_VIOLENT

Description: Violent Crime Rate per 1,000 Population

Property Crime (count)

Data Set: Tract

Variable: COUNT_PROPERTY

Description: Number of Property Crimes

 

Property Crime (per 1,000 population)

Data Set: Tract

Variable: PT_PROPERTY

Description: Property Crime Rate per 1,000 Population

 

Education Score

Data Set: Tract
